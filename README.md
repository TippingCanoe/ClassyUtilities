# Classy Utilities

Helper categories for use in Classy
Let's you specify the use of localized text content, and setting of fonts on UIButtons

```CSS
^UIButton.back{
    cas_font:refash 20;
    cas_localizedTitle:'Go Back Fool!';
}

^UILabel.default{
    cas_localizedText:'I am a Label!';
}
```
